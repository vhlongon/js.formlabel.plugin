JS formLabel plugin

A simple Javascript only (dependency-free) plugin to add a toggle-like visibility effect for form input labels and placeholders

Options:


            defaults = {
                initClass: 'js-formlabel-plugin',
                callbackBefore: function () {
                },
                callbackAfter: function () {
                },
                scope: document,
                selector: 'input[type="text"], input[type="email"]',
                focusColor: '#0eaee8',
                textColor: '#ffffff',
                speed: '.3',
                customStyles: false,
                height: '25px',
                focusClass: 'js-formlabel-plugin-focus',
                blurClass: 'js-formlabel-plugin-blur'
            }
